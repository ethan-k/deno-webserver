# deno-webserver



## Simple Server with std library

```
$ deno run --allow-net --allow-read main.ts
$ curl localhost:8000                               
Hello from your first Deno server%
```


## Simple Server with [Oak](https://github.com/oakserver/oak)

```
cd ./oak
$ deno run --allow-net --allow-read main.ts
$ curl localhost:8000                               
Hello World!
```
